Type and Replace: Maltese

This is a page making it easier to type characters required for typing Maltese. This page is intended to be useful when a person wants to type Maltese but does not have the correct keyboard setup. When you do not have a special keyboard layout, you can come to this page, read the legend on this page, type text in the textbox, and press the button labelled "Replace".

This page can be downloaded, and after downloading can run on a computer even without internet.
