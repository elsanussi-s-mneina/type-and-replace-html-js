Type and Click: Buckwalter Romanization (see Qamus.org)

This is a page making it easier to type characters required for typing Buckwalter Romanization (see Qamus.org) . This page is intended to be useful when a person wants to type Buckwalter Romanization (see Qamus.org) but does not have the correct keyboard setup. When you do not have a special keyboard layout, you can come to this page, and press the buttons to type characters.

This page can be downloaded, and after downloading can run on a computer even without internet.
